What is Blindtrainer and how should I use it?
BlindTrainer is a practice tool for blindfolded Rubik’s Cube solving. Instead of only encountering cases randomly in solves, you can deliberately practice specific commutators, algorithms, and letter pairs. This allows faster recall, better consistency, and more confidence in real solves.

How to practice:
  • Select the category and cases you want to practice
  • Start and complete the practice session
  • Review your results
  • Repeat
What is a practice session?
A practice session consists of a sequence of selected cases randomly shown one by one.

For each case:
  • You recall and execute the solution (or letterpair word)
  • Your time is measured
  • The result is stored for statistics
After the session, you will land on a page with useful statistics to help you improve.
How to use presets?
Presets allow you to save groups of cases for quick reuse. They are useful for cases you practice frequently.

How to use presets:
  • Select the cases you want
  • Click Set next to a preset slot
  • Click the preset to reselect those cases later

Additional actions:
  • Double-click a preset to rename it
  • Click Reset to clear its contents
Presets are stored locally in your browser.
What is the slow preset?
The slow preset is automatically generated based on your practice performance. It contains the cases that took you the longest to execute.

Key points:
  • Tracked separately for each piece type and buffer
  • Updated automatically after every session
  • Helps target slow or inconsistent cases

You can configure how many cases are stored and reset all slow presets in the main settings.
What do the practice settings mean?
Practice settings control how sessions behave:

Cases before break: Number of cases before a break (0 = no breaks)
Target time: Desired maximum time per case. Slower cases can be re-practiced
Amount of cases: Maximum number of cases stored in the slow preset
Max time: Upper time limit for slow cases (prevents storing outliers)
Include inverse cases: Selecting AG will also include GA
Show next letterpair: Displays the upcoming case to better simulate real solves
How do custom letter schemes work?
You can define a custom letter scheme in the main settings.

There are two input fields:
  • Corner letters
  • Edge letters
Enter exactly 24 letters per field, without spaces. The order must follow the Speffz A–X scheme.

Corner scheme is used for:
Corners, Twists, parts of Parity, LTCT, Words, X-centers

Edge scheme is used for:
Edges, parts of Parity, Wings, Midges, +centers

Speffz order:
Corners: UBL UBR UFR UFL LUB LUF LDF LDB FUL FUR FDR FDL RUF RUB RDB RFD BUR BUL BDL BDR DFL DFR DBR DBL
Edges: UB UR UF UL LU LF LD LB FU FR FD FL RU RB RD RF BU BL BD BR DF DR DB DL
How do I change buffers and buffer order?
Buffers are configured in the practice settings.

  • Double-click a buffer to select it
  • Drag and drop buffers to change their order
Each piece type and buffer combination is tracked separately.
How does data export and import work?
BlindTrainer allows you to sync your data using the cloud icon on the landing page.

  • Export: Saves your local data to the database
  • Import: Loads data from the database into your browser

IMPORTANT:
Importing or exporting will completely overwrite existing data. This action cannot be undone.

Your data is linked to a unique ID. Using the same ID on multiple devices allows manual syncing between them. No data will be imported or exported unless you manually do so.
Why is the session chart not visible?
On smaller screens, the session chart is hidden due to limited horizontal space.
